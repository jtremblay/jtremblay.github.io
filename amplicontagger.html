AmpliconTagger

This is the AmpliconTagger home page.

The AmpliconTagger project holds in two code repositories:

  • CentOS 7 Docker image with a functional implementation of the pipelne .
  • One holding the code of the pipeline wrapper .
  • One for the scripts it relies on.
  • An additional repository contains the Third software packages module install scripts.
  • The AmpliconTagger user guide.
  • Databases used in AmpliconTagger - DOI:10.5281/zenodo.3560150.
